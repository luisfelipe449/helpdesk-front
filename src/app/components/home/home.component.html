Help Desk Desenvolvido por Luis Sant'Ana

Olá! Seja bem vindo ao sistema Help Desk desenvolvido por mim.
Esse é um sistema para gerenciamento de chamados simulando um simples Help Desk desenvolvido em Angular 13 e Spring Boot 2.
O sistema está rodando na nuvem sendo o front na Vercel e o back na Heroku.
O projeto tem como objetivo principal abordar os conceitos de aplicações SPAs, Autenticação e Autorização com Tokens JWT e muito mais. O código fonte do projeto pode ser acessado clicando no link Github na barra de navegação.

Fique a vontade para testar o sistema e deixar sua sugestão.
